# Data-Driven Flight Predictive Modeling & Interactive Analytics
### IBM Data Science Professional Capstone Project

## 📌 Project Overview
This repository contains the end-to-end data science capstone project focused on predictive modeling, interactive dashboards, and exploratory data analysis. The project demonstrates how advanced data science methodologies can be deployed to predict operational outcomes, estimate asset utilization costs, and analyze geospatial transport dynamics. 

By mapping rocket launch parameters to financial and operational risks, this repository showcases data workflows directly transferable to aviation asset management, fleet tracking, and leasing risk assessment.

## 🛠️ Tech Stack & Key Deliverables
* **Data Ingestion & Engineering:** Automated REST API integration and automated Web Scraping (`BeautifulSoup`, `requests`).
* **Database Management & EDA:** SQL querying for comprehensive exploratory data analysis, filtering metrics, and database profiling.
* **Interactive Visual Analytics:** Location intelligence and geospatial clustering mapping using `Folium`.
* **Dynamic Dashboards:** Operational telemetry reporting built using `Plotly Dash`.
* **Predictive Modeling:** Multi-algorithm Machine Learning classification (`Logistic Regression`, `SVM`, `Decision Tree`, `KNN`) optimized via `GridSearchCV`.

## 📁 Repository Structure
* **`Week_#1_Data Collection API_.ipynb`**: REST API ingestion and initial payload data framing.
* **`Week_#1_Data Collection Web Scraping_.ipynb`**: Automated extraction of historical launch data from web tables.
* **`Week_#2_EDA_SQL_.ipynb`**: Relational database operations to identify performance, mass anomalies, and success metrics.
* **`Week_#3_Interactive Dashboard with Ploty Dash_.py`**: Script for launching a web-based performance monitoring dashboard.
* **`Week_#3_Interactive Visual Analytics Folium_.ipynb`**: Geospatial launch site mapping and proximity analysis.
* **`Week_#4_Machine Learning Prediction_.ipynb`**: Core predictive modeling pipeline achieving peak classifier accuracy.
* **`Week_#5_SpaceX_Final Presentation_AB__.pdf`**: Executive summary slide deck detailing findings and strategic recommendations.

## 📈 Cross-Industry Value (Aviation Leasing Transferability)
The algorithms and frameworks utilized across these modules simulate core requirements in **Aviation Technical Asset Management**:
1. **Predictive Modeling:** The machine learning pipelines match workflows used to estimate aircraft residual value and component degradation schedules.
2. **Geospatial & Dashboarding:** Folium mapping and Plotly Dash applications demonstrate the exact infrastructure required to track global leased asset distributions, operator routes, and regional risk factors.
3. **Structured Data Audits:** The rigorous data cleaning and SQL architectures parallel the systems used to audit complex aircraft digital record transitions during redelivery cycles.
